# Vercel Express API

A simple Express.js API deployed on Vercel.

## Features

- ✅ Express.js server
- ✅ RESTful API endpoints
- ✅ Static file serving
- ✅ Vercel deployment ready
- ✅ Health check endpoint

## API Endpoints

- `GET /` - Root endpoint with welcome message
- `GET /api/users` - Get sample users data
- `GET /api/health` - Health check endpoint

## Local Development

1. Install dependencies:

```bash
npm install
```
